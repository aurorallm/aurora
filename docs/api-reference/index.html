Chat Completions

POST/v1/chat/completions

Primary inference endpoint. Routes to the optimal provider based on your configured strategy — latency, cost, or primary/fallback.

Parameters

modelRequired

Model ID to use. e.g. gpt-4o, claude-3-opus, aurora-best-latency

Type: string
messagesRequired

Array of message objects representing the conversation.

Type: array
streamOptional

Enable server-sent event streaming of partial responses.

Type: boolean
max_tokensOptional

Maximum tokens in the generated response.

Type: integer
temperatureOptional

Sampling temperature 0–2. Defaults to 1.

Type: number
cURL Example
curl -X POST https://api.aurora.gateway/v1/chat/completions \
  -H "Authorization: Bearer $AURORA_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "Explain quantum computing."}
    ],
    "stream": true
  }'
Response Sample
{
  "id": "aur-8f3a-x1b2",
  "object": "chat.completion",
  "created": 1749268800,
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Quantum computing leverages qubits ..."
      },
      "finish_reason": "stop"
    }
  ],

    "prompt_tokens": 28,

POST/v1/chat/completions?stream=true

Same as the standard endpoint but returns a Server-Sent Events stream. Each chunk contains a delta of the response.

Parameters

modelRequired

Model ID to use.

Type: string
messagesRequired

Conversation messages array.

Type: array
streamRequired

Must be set to true.

Type: boolean
cURL Example
curl -X POST https://api.aurora.gateway/v1/chat/completions \
  -H "Authorization: Bearer $AURORA_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-3-opus",
    "messages": [
      {"role": "user", "content": "Write a short poem."}
    ],
    "stream": true
  }'
Response Sample
data: {"id":"aur-9c2d","choices":[{"delta":{"role":"assistant"}}]}
data: {"id":"aur-9c2d","choices":[{"delta":{"content":"Here"}}]}
data: {"id":"aur-9c2d","choices":[{"delta":{"content":"'s a"}}]}
data: {"id":"aur-9c2d","choices":[{"delta":{"content":" poem..."}}]}
data: [DONE]
