Details

Standard inquiries regarding deployment logistics and billing cycles.

Does OSS include semantic caching?

Yes. All caching capabilities—including exact match and vector-based semantic caching—are available in the Open Source distribution for local or self-hosted vector stores.

What happens if I hit a gated route?

In the OSS profile, enterprise routes return a 403 Forbidden status code. The gateway continues to perform core inference routing and passthrough functions without interruption.

Enterprise Support availability?

Enterprise licenses include 24/7 dedicated engineering support, custom SLAs, and architecture review sessions to ensure maximum availability in mission-critical environments.

How does Aurora compare to LiteLLM or Portkey on pricing?

Aurora OSS is free (Apache 2.0) and self-hosted — no per-request fees, no log limits, no hidden costs. LiteLLM is also free (MIT) but requires heavier infrastructure (3-node Kubernetes vs Aurora's single VM). Portkey is free to start, then $49/month for 100K logged requests. Aurora's Go binary uses 85% less compute than Python-based gateways, so your infrastructure cost is typically lower.
